# VisionGuard Web Deneyimi 👁️

Bu depo, **VisionGuard AI** projesinin web tabanlı simülasyonunu içermektedir. Bu arayüz, görüntü işleme mantığının tarayıcı teknolojileriyle nasıl çalıştığını göstermektedir.

## ✨ Demo Hakkında
Bu çalışma, ana Python uygulamasının temel yüz takip özelliklerini tarayıcı üzerinde simüle eden hafif (lightweight) bir arayüzdür. **Tracking.js** kütüphanesi kullanılarak geliştirilmiştir. Kullanıcıların herhangi bir kütüphane kurmadan projeyi deneyimlemesini sağlar.
